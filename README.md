# Review Sentiment Pro Deployment Guide

Follow these steps to deploy your Restaurant Review Sentiment Analyzer to the web for free.

## Step 1: Create a GitHub Repository
1. Log in to your [GitHub account](https://github.com/).
2. 2. Click the **+** icon in the top right and select **New repository**.
   3. 3. Name it `restaurant-review-sentiment`.
      4. 4. Keep it **Public** (required for the free tier of Streamlit Cloud).
         5. 5. Click **Create repository**.
           
            6. ## Step 2: Upload Files to GitHub
            7. You can use the GitHub website to upload the files:
            8. 1. In your new repository, click **uploading an existing file**.
               2. 2. Drag and drop these files from your local folder:
                  3.    - `app.py`
                        -    - `requirements.txt`
                             -    - `Restaurant_Reviews.tsv`
                                  - 3. Click **Commit changes**.
                                   
                                    4. ## Step 3: Deploy to Streamlit Cloud
                                    5. 1. Go to [share.streamlit.io](https://share.streamlit.io/).
                                       2. 2. Connect your GitHub account.
                                          3. 3. Click **Create app**.
                                             4. 4. Select your repository (`restaurant-review-sentiment`), the branch (`main`), and the main file path (`app.py`).
                                                5. 5. Click **Deploy!**
                                                  
                                                   6. Your app will be live in a few minutes!
                                                   7.
